# Zyloris Brand Theme

## Core Identity
**"Industrial Sovereignty"**
Zyloris is not a SaaS; it is a permanent digital utility. The design language reflects huge, physical infrastructure rather than ephemeral software.

## Visual Language
- **Typography**: `Inter` for interface, `JetBrains Mono` for data/technical details.
- **Colors**:
  - `Background`: #050505 (Deep Void)
  - `Accent`: #3b82f6 (Industrial Blue - used sparingly)
  - `Text`: White (#FFFFFF) & Grey (#888888)
- **Motifs**:
  - Technical markings (grids, crosshairs)
  - Data visualizations (metrics, tickers)
  - Brutalist layout (hard lines, high contrast)
  - Reveal animations (slow, heavy, like hydraulics)

## Philosophy
- **Reliable**: Feel solid and unspoken.
- **Universal**: Accessible to all, owned by none.
- **Permanent**: Built to last forever.
